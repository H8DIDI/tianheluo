关于萤合落 — 烟花燃放控制与3D编排系统

萤合落（Yingheluo）是一款开源的专业烟花燃放控制与3D编排系统，旨在将传统的花炮行业带入数字化时代。项目名称取自"萤火"与"合落"的意象，寓意将点点星火汇聚为壮观的空中盛宴。

🎯 项目背景

中国是烟花的发源地，浏阳作为世界烟花之都，其花炮产业年产值超过508亿元。然而传统的烟花秀设计仍高度依赖人工经验，缺乏可视化编排工具。萤合落正是为解决这一痛点而生。

本项目源于对文化创意赋能花炮产业的深度思考——如何用数字技术提升传统产业的价值链？烟花编排系统就是答案之一。

🏗️ 技术架构

萤合落采用现代 Web 技术栈构建，完全在浏览器中运行，无需安装任何软件：

  • 前端框架：React 19 + TypeScript + Vite 6
  • 3D渲染引擎：Three.js — 使用 InstancedMesh 实现50000粒子的实时渲染
  • 物理引擎：自研二次方阻力模型，模拟真实的烟花弹道轨迹
  • 发色模型：基于金属盐冷却发色原理，还原真实烟花颜色变化
  • 状态管理：Zustand — 轻量高效的 React 状态管理
  • 后端：Cloudflare Workers + D1 数据库
  • 部署：Cloudflare Pages — 全球 CDN 加速

💡 设计理念

萤合落的设计理念围绕三个核心：

  • 专业性：800管阵地布局、多种发射模式、对称齐射编排，满足专业烟花秀设计需求
  • 易用性：AI智能编排助手，用自然语言描述需求即可生成烟花秀方案
  • 真实性：物理引擎模拟真实弹道，金属盐发色模型还原烟花颜色
