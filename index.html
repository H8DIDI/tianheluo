🎆 天河落 — 浏阳花炮产业聚合站

天河落（Tianheluo）是聚焦中国浏阳花炮产业的数字化聚合展示站， 通过自建数据管线日更来自 B 站、YouTube 的专业烟花视频，追踪浏阳花炮相关资讯， 呈现完整产业链与关键数据，并提供 3D 烟花编排、预算测算等实用工具集。

📺 五个主要板块

  • 视频墙 — B站 / YouTube 浏阳烟花与烟花秀专业内容聚合，日更，标签筛选
  • 产业链 — 上游原料装备 · 中游药剂组装 · 下游流通应用，一屏看全景
  • 资讯 — 百度新闻、财经媒体、政府公众号关键词追踪，仅标题摘要跳转原站
  • 数据 · 企业 — 产业指标 + 规上企业名录（API 对接中）
  • 工具集 — 3D 烟花编排模拟器、燃放时长估算、安全距离计算器、金属盐发色对照等

🎇 浏阳花炮 — 世界烟花之乡

浏阳位于中国湖南，是世界烟花之乡，花炮产业已有 1400 多年历史，是全球最大的烟花生产基地。

浏阳花炮产业年产值达 508.9 亿元，拥有 431 家规上企业和 3721 项相关专利， 产品出口 100 多个国家。在文化创意赋能背景下，浏阳正从传统花炮制造向数字化、 智能化转型，天河落正是服务于这一转型的聚合展示窗口。

🔧 技术架构

前端：React 19 + TypeScript + Vite 6 + Tailwind + React Router
后端：Cloudflare Pages Functions + D1 数据库
数据管线：阿里云 ECS 上 RSSHub + Python 抓取脚本 + systemd timer 日更
分发：Cloudflare 全球 CDN + Cloudflared Tunnel
